# DA-internship-infotact-Finance---Market-Risk-Volatility-Dashboard
A Python-based quantitative finance project that analyzes stock market risk using daily returns, rolling volatility, Monte Carlo simulation, and Value at Risk (VaR). Includes an interactive Streamlit dashboard for visualization.

🚀 Features

Historical stock data analysis

Daily return calculation

20-day rolling volatility

Monte Carlo price simulation

Expected future price forecast

Value at Risk (95%) estimation

Interactive Streamlit dashboard

🧠 Techniques Used

Quantitative Risk Modeling

Monte Carlo Simulation

Statistical Volatility Analysis

Probability-based Forecasting

🛠 Tech Stack

Python

Pandas

NumPy

Matplotlib

Streamlit

▶️ Run Locally
pip install pandas numpy matplotlib streamlit
streamlit run dashboard.py

Financial risk analysis

Portfolio risk monitoring
